# 《置身事内》知识 Wiki

## 书籍概述

- **书名**：置身事内——中国政府与经济发展
- **作者**：兰小欢
- **主题**：中国政府与经济发展的关系

## 知识库结构

```
references/
  transcripts/     # 妙计语音转写 .md 文档（按章节组织）
  book/           # PDF / EPUB 原书
  notes/          # 人工整理的笔记、摘要
```

## 使用方式

1. **知识问答**：直接提问关于本书内容的问题，基于 references/ 中的原文回答
2. **读取文档**：将新的转写文档或书籍文件放入 references/ 对应目录
3. **章节梳理**：按需整理特定章节的核心论点、时间线、因果关系
4. **专题归纳**：将分散在各章节中关于同一主题的内容聚合

## 回答原则

- **基于原文**：引用原始文档中的具体表述，不自行推断
- **注明来源**：每次回答标注来源文件和时间戳
- **不添加外部内容**：只基于本书内容回答，不引入未在 references/ 中的外部知识
- **原文整理**：不添减内容，不加对比表、分析框架、术语解释
