# Skills

为 Hermes Agent 编写的自定义 skill，存放于 `~/cakemonster/skills/`，软链接到 `~/.hermes/skills/trading/` 供 Agent 加载。

---

## bilibili-download

B站音视频下载器，通过 B站公开 API 获取 DASH 流，无需登录、无需 cookies。

**功能：** 音频提取（m4s → mp3）、视频提取（m4s → mp4）、完整视频下载（音视频合并）

---

## github-trending

抓取 GitHub Trending 日/周/月榜单，基于 BeautifulSoup 解析 SSR HTML。

**功能：** 日榜 / 周榜 / 月榜 三榜抓取，输出 Markdown 格式（含项目名、语言、star 数、今日新增）

---

## xhs-scraper

小红书通用抓取器，基于 Playwright CLI 浏览器自动化。

**功能：** QR 码登录、Cookie 持久化、博主主页笔记抓取、单篇笔记正文抓取

---

## benjiu-trading-system

笨笨的高景气价值投机体系——基于现象级事件驱动的产业拐点选股法。

**核心方法：** 现象级事件四要素/五要素鉴别框架、拐点事件四分类、选股四步法、牛股七维打分表、笨韭双击/单击、剑宗气宗双模式操作。

**定位：** 不做超短、不做传统价投、不做纯概念博弈——在产业爆发前夜布局最受益标的，让机构来抬轿。

---

## commodity-data-source

A 股产业链原材料数据源——百川盈孚产业链 + 国内期货品种实时行情。

**核心数据：** 百川盈孚（baiinfo.cn）9 大产业频道约 300 个品种的产业链上下游链路关系；国内期货 63 个品种实时行情（mootdx）。

**用途：** 为日晷（Sundial）产业链图谱 macro-industry.json 提供原材料节点定义与价格接入。铁律：禁止 mock 数据，数据源失败返回空。

---

## market-sentiment

A 股市场情绪量化判断——四层框架 + 五阶段输出。

**四层框架：** 宏观水温 → 短线热度 → 赚钱/亏钱效应 → 主线方向

**五阶段：** 冰点（≤20%仓）/ 修复（30-50%）/ 分歧（50-70%）/ 高潮（70-100%）/ 退潮（≤30%）

**数据源：** sundial dashboard + 淘股吧热帖 + 同花顺热榜

---

## sundial-dashboard

日晷（Sundial）—— A 股市场数据看板，FastAPI SPA，端口 8100。

**核心页面：** 每日复盘（情绪+连板天梯）、热榜（历史+实时）、个股分析（K线+分时+动态队友+大单异动）、策略回测、模拟账户、产业链图谱。

**数据源：** 东方财富（涨停/炸板池）、Baostock（指数+PE）、同花顺（热榜+板块）、mootdx（个股K线+分时+逐笔）、Sina（实时涨跌幅补全）。

**特色功能：** 队友量化（概念分组+1分钟K线滑动Pearson r互相关）、大单异动（逐笔成交≥1000万持久化SQLite）、T+1/涨跌停/手续费引擎（已合并入sundial进程内）。

**零 mock 兜底：** 数据源失败返回空，不生成假数据。
