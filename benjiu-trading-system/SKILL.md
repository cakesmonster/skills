# 笨笨的韭菜 · 高景气价值投机体系

> 基于现象级事件驱动的产业拐点选股法。不做超短、不做传统价投吃息、不做纯概念博弈——只在产业爆发前夜，找到最受益的标的。

---

## 核心理念：高景气价值投机

A股散户常见的三种流派：

| 流派 | 特点 | 缺陷 |
|------|------|------|
| 超短（看线） | 纯技术面，不看基本面 | 零和博弈，胜率依赖市场环境 |
| 超短（消息） | 比手速，爬虫抢消息差 | 需要极低延迟，普通散户无法做到 |
| 价投/吃息 | 低估值+高分红，长坡厚雪 | 适合超大资金（千万级以上），小资金效率极低 |

**第四种——高景气价值投机（高景气价投/投机）**：

- 本质是**前置布局**：在产业拐点出现但业绩尚未兑现时介入
- 不与机构比PE/PB——机构等业绩兑现后才进场，他们是来抬轿的
- 核心逻辑：**现象级事件 → 产业拐点 → 高景气赛道 → 精选标的 → 等待爆发**

## 一、选股四步法

```
行业机会发现 → 板块（细分领域）确认 → 标的筛选 → 风险识别
```

### 1.1 行业机会发现

**两条路径**：

**路径A：现象级事件带来的拐点（时效性强）**
- 足以改变一个板块/细分领域估值的大事件
- 详见「三、拐点事件四分类」「二、现象级事件鉴别框架」

**路径B：寻找高景气行业（可持续关注）**
必须同时满足以下四点：
1. 有政策+资金双支持
2. 市场足够大，未来几年确定全民使用（如AI、商业航天、智能驾驶）
3. 商业化来到临界点/初期（有企业开始赚钱）
4. **反面筛选**：规避氢能源、超导、核聚变——技术未突破、商业化遥遥无期

**传统行业拐点需满足多个条件**：
- 跌得够多 + 跌的时间够长 + 跑完整个周期
- 行业大量倒闭/破产/退市（产能出清）
- 未来需求稳定且最好有增长
- 技术突破使商业价值骤增

### 1.2 板块（细分领域）确认

**新兴科技领域**：选**最上游、最有技术壁垒、无法绕过**的环节
- 案例：AI → GPU算力卡（寒武纪）> 光模块 > 下游应用

**传统行业拐点**：选**最下游、最强**的头部企业
- 原因：经历过完整周期后，定价权在下游品牌方，上游已被卷烂

### 1.3 标的筛选

四大要素（详见「四、牛股筛选打分表」）：
1. **业务纯度**：看上那块业务占营收/利润比例越高越好
2. **历史股价低位**：近3年最低价涨幅<20%=满分
3. **细分行业龙头**：全球龙头>国内龙头>A股龙头
4. **市场辨识度**：名字、稀缺性（两市唯一标的=极高辨识度）

### 1.4 风险识别

**原地坐牢风险**（买入后不涨）：
- 传统行业整体估值偏低压制
- 股东套现/减持历史
- 业绩迟迟无法兑现（预期明年变3年）

**暴跌风险**（黑天鹅）：
- 内部暴雷：年报保留意见、财务造假前科、被ST
- 安全事故：矿难、服务器宕机、生产线事故
- 外部突发事件：美国制裁（议员提案）、友商技术突破/产能爆发

**一票否决**：触及国家底线红线、禁令——任何情况下不参与，等整改完成再考虑小仓位博弈。

## 二、现象级事件鉴别框架

### 2.1 现象级事件定义

**现象级事件（拐点事件）**：发生后能够明显改变一个行业/个股的景气程度，触发二级市场对该细分板块进行价值重估，开启主升浪行情。

### 2.2 突发新闻事件 — 四要素（一票否决制）

| 要素 | 说明 | 权重 |
|------|------|:--:|
| **真实性** | 必须交叉验证，假消息=后面所有逻辑全废 | ⭐⭐⭐ 一票否决 |
| **传播性** | 范围要大，最好出圈形成全民讨论 | ⭐⭐ |
| **规模性** | 涉及行业体量要大，有足够想象空间 | ⭐⭐⭐ |
| **时效性** | 第一次最有价值，重复消息情绪递减 | ⭐⭐ |

**四要素实战对照**：
| 事件 | 真实性 | 传播性 | 规模性 | 时效性 | 结论 |
|------|:---:|:---:|:---:|:---:|------|
| ChatGPT（2023.1） | ✅ | ✅ 现象级传播 | ✅ 全行业可改造 | ✅ 首次 | **满分** |
| 芯片制裁（2018） | ✅ | ✅ 全民讨论 | ✅ 中国制造业全上游 | ✅ 首次 | **满分** |
| 尊界S800智驾 | ✅ | ✅ 华为流量 | ❌ 超豪华年销≤1万台 | ✅ 首次 | **规模不够→pass** |
| 锂硫电池突破 | 算你真实 | ❌ 几乎没人讨论 | — | — | **离落地太远+非主流国家科学家→pass** |
| 四川产业链搬迁 | ❌ 第二天辟谣 | ✅ 有人花钱推 | — | — | **假消息→一票否决** |

### 2.3 国内政策新闻事件 — 五要素

政策新闻分三类（层级递减）：
1. **顶层设计/规划**（政治局级别）→ 以口号为主，无具体资金规模
2. **国务院/部委政策** → 详细政策+资金规模+覆盖群体——重点关注
3. **地方政策** → 规模太小，一般不构成现象级事件

| 要素 | 说明 |
|------|------|
| **落地周期** | 有没有明确落地时间？「未来几年」=慎重 |
| **数字规模** | 官方投多少钱？（最重要！） |
| **持续时间** | 一锤子刺激还是长期？ |
| **覆盖群体规模** | 全民还是局部？ |
| **时效性** | 天天喊=无效；形容词政策→直接pass |

**核心原则**：看钱 → 看时间 → 看规模。没有具体数字的口号政策，直接忽略。

## 三、拐点事件四分类

| 类别 | 说明 | 案例 |
|------|------|------|
| **科技突破** | 某领域技术/应用出现重大突破 | GPT（2023.1）、iPhone4（2010）、特斯拉Model S量产（2013） |
| **政治经济** | 贸易战、加关税、芯片制裁等政策事件 | 2019半导体制裁→国产芯片起飞 |
| **自然灾害/军事冲突** | 导致矿产/农产品全球供应大幅萎缩 | 俄乌战争→小麦 |
| **海外映射** | 海外企业跑通模式、业绩爆发，国内对标价值重估 | AI广告（AppLovin业绩爆炸→国内AI广告映射） |

**伪现象级事件识别**：
- 部委/地方政府喊话支持某行业（冲一下就没）
- 某企业宣布「全面进入XX行业」（除非行业刚大爆发+第一家）
- 板块所有标的狂飙+头部业绩超预期→往往是出货信号

## 四、牛股筛选打分表

> **大前提**：必须先确认行业存在超景气机会，否则表格无效。

### 打分维度（每项0-100分）

| 维度 | 评分标准 | 权重 |
|------|----------|:--:|
| **行业景气度** | 二级市场景气，现象级事件→100分 | 高 |
| **业务纯度** | 标的业务占营收/利润比例，越高越好；海外业务≥50%=80分起步 | **最高** |
| **历史估值位置** | 近3年最低价涨幅≤20%→100分，每涨1%扣1分 | 中 |
| **市值空间 / 细分龙头** | 全球龙头100分；国内龙头且海外竞品进不来→100分 | 高 |
| **市场辨识度** | 名字含行业关键词、两市稀缺唯一标的 | 中 |
| **管理层 / 风险扣分** | 解禁、减持、黑天鹅官司→扣分项 | 扣分 |
| **技术壁垒** | 卡位股、无法绕过的技术环节优先 | 高 |

### 流动性系数（乘数）

| 全市场成交额 | 系数 |
|-------------|:---:|
| ≤0.8万亿 | ×0.8 |
| 1.0万亿 | ×1.0 |
| ≥1.5万亿 | ×1.2 |

### 打分公式

```
总分 = (各项分数 × 权重求和 - 风险扣分) × 流动性系数
>80分 = 开门（可直接买入）
```

## 五、笨韭双击 vs 传统戴维斯双击

### 传统戴维斯双击
- 在**低PE**时买入，待公司成长潜力显现后以**高PE**卖出
- **局限**：仅适用于周期性传统行业；科技行业前期爆亏，没有PE可言

### 笨韭双击
> **产业拐点 + 企业基本面拐点同时出现**
- 产业端：现象级事件导致行业拐点
- 企业端：公司提前于行业出现业绩拐点（营收持续增长、亏损收窄、新业务突破）
- **全程忽略PE/PB**

### 笨韭单击
> 只有产业拐点，企业基本面尚未改善（更常见）
- 可能因为预期差更大，涨幅反而更高

### 案例
| 类型 | 标的 | 产业拐点 | 企业拐点 |
|------|------|----------|----------|
| 笨韭双击 | 曙光数创（23Q1） | ChatGPT→智算中心需求爆发 | Q1业绩爆单 |
| 笨韭双击 | 速腾聚创（24Q3） | 智能驾驶爆发前夜 | 营收+利润同比持续大涨 |
| 笨韭单击 | 海能达 | 中东BB机爆炸→对讲机需求爆发 | 无业绩拐点 |

## 六、A股上涨三因素

| 因素 | 特征 | 适用场景 |
|------|------|----------|
| **流动性溢价** | 指数普涨，风险偏好提升，小票弹性大 | 牛市 |
| **高股息抱团** | 防守型资金，熊市抗跌 | 熊市/震荡市 |
| **消息事件驱动** | 现象级事件→行业拐点（本体系主攻方向） | 全年皆可 |

## 七、剑宗（一波流）vs 气宗（长期格局）

| 维度 | 剑宗（一波流） | 气宗（长期格局） |
|------|:---:|:---:|
| 适用事件 | 海外映射、其他现象级事件 | 0→1超级事件、1→10产业化突破 |
| 业绩支撑 | 短期无业绩 | 业绩持续显现 |
| 持有周期 | 数周到一年 | 长期，可扛30-40%回调 |
| 仓位目标 | 30% | 70% |
| 止盈信号 | 破线即走，不格局 | 短期暴涨5-10倍必须止盈 |

### 核心纪律

1. **逻辑崩塌立刻卖出**
2. **不与股票谈恋爱**：格局票短期暴涨5-10倍→必须止盈
3. **扛得住回调**：基于产业逻辑买入，短期-40%不走，除非逻辑崩塌
4. **流动性是最后保险**：大盘连续破1.3万亿=流动性风险信号
5. **不追高**：板块所有标的都在主升浪狂飙+头部业绩超预期→出货信号
6. **不畏高PE**：科技行业早期爆亏，PE/PB无参考意义
