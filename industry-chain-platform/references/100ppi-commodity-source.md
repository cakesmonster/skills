# 生意社（100ppi.com）大宗商品价格数据源

## 定位

生意社是中国大宗商品现货报价的权威平台。与期货不同，它覆盖了**非期货化**的品种（萤石、稀土氧化物、电子特气等），是产业链原材料清单的第二个锚（第一个锚是期货品种列表）。

## 品种目录

9 大分类，总计 5000+ 品种。与产业链图谱相关的主要分类：

| 分类 | 品种数 | 图谱相关品种示例 |
|------|:---:|------|
| 能源 | 59 | 原油、液化天然气、焦炭、动力煤、无烟煤 |

## 反爬与绕过

网站使用 **HW_CHECK JS Challenge**：返回一段 JavaScript 计算 cookie 值，未执行 JS 的请求（curl/wget）永远被拦。

**解决方案**：使用 Playwright 浏览器。Playwright 自带 JS 执行环境，会自动通过 HW_CHECK，无需手工处理 cookie。

```
# 点击类操作：browser_navigate → browser_click
# 数据提取：browser_console 执行 JS 提取 DOM 内容
# 注意：不要用 curl/scrape 直接请求，必须走浏览器
```

## 价格页面

每个品种有独立的价格详情页，URL 格式：`https://www.100ppi.com/price/detail-{品种ID}.html`

品种 ID 可以从报价中心页面的链接中提取（href 格式：`plist-1-{id}-1.html`）。

## 接入方案

1. **提取品种目录**：Playwright → 报价中心 → 按分类抓品种名和 ID
2. **映射到图谱节点**：人工维护"原材料→节点"映射表（见 SKILL.md 待办部分）
3. **抓取价格**：按需抓取特定品种的当前报价和涨跌幅
4. **叠加到图谱**：前端 hover tooltip 显示关联原材料价格

## 注意事项

- 化工类 4774 品种太多，需要按图谱节点筛选出"大宗商品级"品种（过滤精细化工品）
- 品种名可能有多种别名，映射需要人工校对
- 价格单位不统一（元/吨、元/千克、美元/吨等），前端需要标注单位
